# Sabores Royal

Aplicación web en un solo archivo HTML para la tienda Sabores Royal, con catálogo, carrito, checkout y vista de administración.

## Cómo abrirlo

Abre el archivo [sabores-royal-completo (11).html](sabores-royal-completo%20(11).html) en tu navegador.

## Subir a GitHub

1. Crea un repositorio nuevo en GitHub.
2. Conecta el repositorio remoto:
   ```bash
   git remote add origin https://github.com/USUARIO/NOMBRE-REPO.git
   ```
3. Sube los cambios:
   ```bash
   git branch -M main
   git push -u origin main
   ```
